# Anonymous Community Map (MVP)

This is a lightweight, privacy-first community mapping tool designed for local use.

The goal is simple:
- Let people share information anonymously
- Avoid accounts, tracking, or long-term data storage
- Automatically remove content after a short time

This project is intentionally minimal.

---

## How it works

- Users can drop anonymous pins on a public map
- Pins may include a short optional note
- Pins automatically expire after a limited time
- No login, no profiles, no identifiers

This is an early MVP built for local pilots.

---

## Privacy principles (English)

We take a **data minimization** approach:

- No user accounts
- No analytics
- No cookies
- No IP addresses stored
- No third-party tracking
- No long-term logs

Pins exist temporarily and are automatically removed.

This project is designed to reduce risk, not extract data.

---

## Principios de privacidad (Español)

Este proyecto sigue un enfoque de **minimización de datos**:

- Sin cuentas de usuario
- Sin analítica
- Sin cookies
